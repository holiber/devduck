# VCS Module

Module providing common rules and conventions for version control systems (VCS).
Contains formatting rules for commits and pull requests that are independent of the specific VCS implementation.

## Purpose

This module provides:
- Commit message formatting rules
- Pull request description formatting rules
- General VCS best practices

## Note

VCS-specific implementations (like `arc` commands for Arcadia) are provided by external modules.
